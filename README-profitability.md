# Profitability-improvement plan

- Gate every trade with explicit expectancy and risk-reward checks.
- Move to true risk-based position sizing (1%-2% risk per trade is a good starting point).
- Use ATR-based dynamic stops and targets to adapt to volatility.
- Subtract estimated fees/slippage in edge calculations.
- Backtest changes on representative historical data; track expectancy, max drawdown, and win rate.
